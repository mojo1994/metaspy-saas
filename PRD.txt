PROMPT TÉCNICO: IMPLEMENTAÇÃO DE QUEBRA DE CLOACKER E CAMUFLAGEM DE CRIATIVOS
1. OBJETIVO GERAL
Adicionar duas novas ferramentas no módulo Cloacker do MetaSpy:

Quebra de Cloacker (Cloak Detector): ferramenta que analisa uma URL enviando requisições com User-Agents distintos (humano x robô) e exibe as diferenças nas respostas (código HTTP, cabeçalhos, redirecionamentos e corpo HTML), indicando se há cloaking ativo.

Camuflagem de Criativos (Creative Camouflage): ferramenta que gera scripts/HTML prontos para ofuscar textos, imagens e CTAs em uma landing page, impedindo que os robôs de revisão do Meta identifiquem o conteúdo publicitário, enquanto usuários reais veem o conteúdo normal.

Ambas as funcionalidades devem ser liberadas exclusivamente para usuários com plano anual.

2. BACKEND (NODE.JS + EXPRESS)
2.1. Novo endpoint: /api/cloaker/detect
Arquivo: server/index.js

Descrição: Recebe uma URL, faz duas requisições paralelas (ou sequenciais) com User-Agent de humano e de robô. Retorna um objeto comparativo.

Script necessário:

javascript
// server/index.js — adicionar após as outras rotas do cloaker

// Lista de User-Agents para teste
const USER_AGENTS = {
  humano: 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/120.0.0.0 Safari/537.36',
  bot_google: 'Mozilla/5.0 (compatible; Googlebot/2.1; +http://www.google.com/bot.html)',
  bot_facebook: 'facebookexternalhit/1.1 (+http://www.facebook.com/externalhit_uatext.php)'
};

// Rota de detecção
app.post('/api/cloaker/detect', authMiddleware, async (req, res) => {
  try {
    // 1. Verifica plano anual
    const features = PLAN_FEATURES[req.user.plan];
    if (!features?.cloaker) {
      return res.status(403).json({ erro: 'Funcionalidade disponível apenas no plano Anual.' });
    }

    const { url } = req.body;
    if (!url) {
      return res.status(400).json({ erro: 'URL é obrigatória.' });
    }

    // 2. Executa requisições para cada User-Agent
    const resultados = {};
    for (const [tipo, ua] of Object.entries(USER_AGENTS)) {
      const controller = new AbortController();
      const timeout = setTimeout(() => controller.abort(), 8000);

      try {
        const resposta = await fetch(url, {
          headers: { 'User-Agent': ua },
          redirect: 'manual',          // captura redirecionamentos
          signal: controller.signal
        });

        clearTimeout(timeout);

        // Lê o corpo (limitado a 50kb para não sobrecarregar)
        const corpoBuffer = await resposta.arrayBuffer();
        const corpo = Buffer.from(corpoBuffer).toString('utf-8').slice(0, 50000);

        resultados[tipo] = {
          status: resposta.status,
          statusText: resposta.statusText,
          urlFinal: resposta.url,
          headers: Object.fromEntries(resposta.headers.entries()),
          corpo: corpo,
          tamanho: corpo.length
        };
      } catch (erro) {
        clearTimeout(timeout);
        resultados[tipo] = {
          erro: erro.message || 'Falha na requisição'
        };
      }
    }

    // 3. Compara os resultados para identificar cloaking
    const temCloaking = (
      resultados.humano?.status !== resultados.bot_google?.status ||
      resultados.humano?.urlFinal !== resultados.bot_google?.urlFinal ||
      resultados.humano?.corpo?.length !== resultados.bot_google?.corpo?.length
    );

    // 4. Salva histórico no banco (opcional, mas recomendado)
    await run(
      `INSERT INTO cloak_detections (id, user_id, url, tem_cloaking, resultado_json, created_at)
       VALUES ($1, $2, $3, $4, $5, $6)`,
      [
        crypto.randomUUID(),
        req.user.id,
        url,
        temCloaking,
        JSON.stringify(resultados),
        new Date().toISOString()
      ]
    );

    res.json({
      url,
      temCloaking,
      comparacao: resultados,
      resumo: temCloaking
        ? '⚠️ Cloaking detectado! A página entrega conteúdo diferente para robôs e humanos.'
        : '✅ Nenhum cloaking significativo detectado.'
    });

  } catch (erro) {
    console.error('Erro no detect cloaker:', erro);
    res.status(500).json({ erro: 'Erro interno ao analisar a URL.' });
  }
});
2.2. Novo endpoint: /api/cloaker/camouflage
Arquivo: server/index.js

Descrição: Gera um script de camuflagem personalizado com base nos parâmetros do usuário (texto a esconder, URL de destino, palavras-chave).

Script necessário:

javascript
// server/index.js — adicionar após a rota de detect

app.post('/api/cloaker/camouflage', authMiddleware, async (req, res) => {
  try {
    const features = PLAN_FEATURES[req.user.plan];
    if (!features?.cloaker) {
      return res.status(403).json({ erro: 'Disponível apenas no plano Anual.' });
    }

    const { texto_original, url_destino, palavras_sensiveis } = req.body;
    if (!texto_original || !url_destino) {
      return res.status(400).json({ erro: 'Texto original e URL destino são obrigatórios.' });
    }

    // Estratégias de camuflagem:
    // 1. Ofuscação de texto com CSS (display:none para bots, visível para humanos)
    // 2. Substituição de caracteres por Unicode similares (ex: "A" por "А" cirílico)
    // 3. Embaralhamento de tags HTML com data-attributes
    // 4. Link duplo: redirecionamento via JS para humanos, link estático para bots

    const scriptCamuflado = `
<!-- SCRIPT DE CAMUFLAGEM DE CRIATIVOS - METASPY -->
<script>
(function() {
  const isBot = /bot|googlebot|facebookexternalhit|headless/i.test(navigator.userAgent);
  const textoOriginal = ${JSON.stringify(texto_original)};
  const urlDestino = ${JSON.stringify(url_destino)};

  if (isBot) {
    // Para robôs: exibe texto genérico ou substitui palavras sensíveis
    const palavras = ${JSON.stringify(palavras_sensiveis || [])};
    let textoMascarado = textoOriginal;
    palavras.forEach(palavra => {
      const regex = new RegExp(palavra, 'gi');
      textoMascarado = textoMascarado.replace(regex, '●●●●●');
    });
    document.body.innerHTML = document.body.innerHTML.replace(textoOriginal, textoMascarado);
    
    // Remove links de rastreio
    document.querySelectorAll('a[href]').forEach(el => {
      if (el.href.includes(urlDestino)) {
        el.href = '#';
        el.style.textDecoration = 'none';
        el.style.color = 'inherit';
      }
    });
  } else {
    // Para humanos: garante que o conteúdo original seja exibido
    // (já vem no HTML, mas garante a visibilidade)
    document.querySelectorAll('.camuflado').forEach(el => el.style.display = 'block');
  }
})();
</script>
<!-- FIM DO SCRIPT -->
    `;

    // Salva no banco (opcional)
    const id = crypto.randomUUID();
    await run(
      `INSERT INTO camouflage_scripts (id, user_id, url_destino, script_code, created_at)
       VALUES ($1, $2, $3, $4, $5)`,
      [id, req.user.id, url_destino, scriptCamuflado, new Date().toISOString()]
    );

    res.json({
      id,
      script: scriptCamuflado,
      instrucoes: 'Copie e cole este script no <head> da sua landing page. O script detectará automaticamente se o visitante é humano ou robô.'
    });

  } catch (erro) {
    console.error('Erro ao gerar camuflagem:', erro);
    res.status(500).json({ erro: 'Erro ao gerar script de camuflagem.' });
  }
});

3. FRONTEND (REACT + TYPESCRIPT)
3.1. Novo componente: CloakDetector.tsx
Arquivo: src/pages/dashboard/CloakDetector.tsx

Descrição: Página com campo de URL, botão "Analisar" e exibição dos resultados (comparação lado a lado).

Script necessário:

tsx
// src/pages/dashboard/CloakDetector.tsx
import React, { useState } from 'react';
import { useAuth } from '../../contexts/AuthContext';

const CloakDetector: React.FC = () => {
  const { fetchWithAuth } = useAuth();
  const [url, setUrl] = useState('');
  const [carregando, setCarregando] = useState(false);
  const [resultado, setResultado] = useState<any>(null);
  const [erro, setErro] = useState('');

  const handleDetectar = async () => {
    if (!url) return;
    setCarregando(true);
    setErro('');
    try {
      const res = await fetchWithAuth('/api/cloaker/detect', {
        method: 'POST',
        body: JSON.stringify({ url })
      });
      const data = await res.json();
      if (!res.ok) throw new Error(data.erro || 'Falha na análise');
      setResultado(data);
    } catch (err: any) {
      setErro(err.message);
    } finally {
      setCarregando(false);
    }
  };

  return (
    <div className="cloak-detector">
      <h2>🕵️ Quebra de Cloacker</h2>
      <p>Descubra se uma página está usando cloaking. Comparamos a resposta para humanos e robôs.</p>

      <div className="input-group">
        <input
          type="url"
          placeholder="https://exemplo.com/pagina"
          value={url}
          onChange={(e) => setUrl(e.target.value)}
        />
        <button onClick={handleDetectar} disabled={carregando}>
          {carregando ? 'Analisando...' : '🔍 Analisar'}
        </button>
      </div>

      {erro && <div className="erro">{erro}</div>}

      {resultado && (
        <div className="resultado">
          <div className="resumo" style={{ background: resultado.temCloaking ? '#ffdddd' : '#ddffdd' }}>
            {resultado.resumo}
          </div>

          <div className="comparacao-grid">
            <div className="coluna humano">
              <h3>👤 Visão Humana</h3>
              <pre>{JSON.stringify(resultado.comparacao.humano, null, 2)}</pre>
            </div>
            <div className="coluna bot">
              <h3>🤖 Visão do Robô (Googlebot)</h3>
              <pre>{JSON.stringify(resultado.comparacao.bot_google, null, 2)}</pre>
            </div>
          </div>

          <details>
            <summary>📄 Ver corpo HTML completo (humano)</summary>
            <div className="html-preview" dangerouslySetInnerHTML={{ __html: resultado.comparacao.humano?.corpo || 'Sem conteúdo' }} />
          </details>
        </div>
      )}
    </div>
  );
};

export default CloakDetector;
3.2. Novo componente: CreativeCamouflage.tsx
Arquivo: src/pages/dashboard/CreativeCamouflage.tsx

Descrição: Formulário para gerar script de camuflagem com campos: texto original, URL destino e palavras sensíveis (separadas por vírgula).

Script necessário:

tsx
// src/pages/dashboard/CreativeCamouflage.tsx
import React, { useState } from 'react';
import { useAuth } from '../../contexts/AuthContext';

const CreativeCamouflage: React.FC = () => {
  const { fetchWithAuth } = useAuth();
  const [textoOriginal, setTextoOriginal] = useState('');
  const [urlDestino, setUrlDestino] = useState('');
  const [palavrasSensiveis, setPalavrasSensiveis] = useState('');
  const [scriptGerado, setScriptGerado] = useState('');
  const [copiado, setCopiado] = useState(false);
  const [carregando, setCarregando] = useState(false);
  const [erro, setErro] = useState('');

  const handleGerar = async () => {
    if (!textoOriginal || !urlDestino) return;
    setCarregando(true);
    setErro('');
    try {
      const res = await fetchWithAuth('/api/cloaker/camouflage', {
        method: 'POST',
        body: JSON.stringify({
          texto_original: textoOriginal,
          url_destino: urlDestino,
          palavras_sensiveis: palavrasSensiveis.split(',').map(p => p.trim()).filter(Boolean)
        })
      });
      const data = await res.json();
      if (!res.ok) throw new Error(data.erro || 'Falha ao gerar');
      setScriptGerado(data.script);
    } catch (err: any) {
      setErro(err.message);
    } finally {
      setCarregando(false);
    }
  };

  const handleCopiar = () => {
    navigator.clipboard.writeText(scriptGerado);
    setCopiado(true);
    setTimeout(() => setCopiado(false), 3000);
  };

  return (
    <div className="creative-camouflage">
      <h2>🎭 Camuflagem de Criativos</h2>
      <p>Gere um script que oculta textos e links sensíveis dos robôs de revisão, mantendo-os visíveis para humanos.</p>

      <div className="form-camouflage">
        <label>
          Texto original (o que será exibido para humanos)
          <input
            type="text"
            value={textoOriginal}
            onChange={(e) => setTextoOriginal(e.target.value)}
            placeholder="Ex: Compre agora com 50% de desconto"
          />
        </label>

        <label>
          URL de destino (link da oferta)
          <input
            type="url"
            value={urlDestino}
            onChange={(e) => setUrlDestino(e.target.value)}
            placeholder="https://exemplo.com/oferta"
          />
        </label>

        <label>
          Palavras sensíveis (separadas por vírgula)
          <input
            type="text"
            value={palavrasSensiveis}
            onChange={(e) => setPalavrasSensiveis(e.target.value)}
            placeholder="comprar, desconto, oferta, grátis"
          />
          <small>Essas palavras serão substituídas por "●●●●●" para robôs.</small>
        </label>

        <button onClick={handleGerar} disabled={carregando}>
          {carregando ? 'Gerando...' : '⚙️ Gerar Script de Camuflagem'}
        </button>
      </div>

      {erro && <div className="erro">{erro}</div>}

      {scriptGerado && (
        <div className="script-resultado">
          <h3>✅ Script gerado com sucesso!</h3>
          <div className="script-box">
            <pre>{scriptGerado}</pre>
          </div>
          <button onClick={handleCopiar}>
            {copiado ? '✅ Copiado!' : '📋 Copiar script'}
          </button>
          <p className="instrucao">
            💡 Cole este script dentro da tag <code>&lt;head&gt;</code> da sua página.
          </p>
        </div>
      )}
    </div>
  );
};

export default CreativeCamouflage;

4. CONSIDERAÇÕES FINAIS
Permissões: Ambas as rotas backend verificam features.cloaker, garantindo que apenas planos anual tenham acesso.

Histórico: As tabelas cloak_detections e camouflage_scripts armazenam o histórico do usuário, permitindo futuramente listar as análises e scripts gerados.

UX: Os componentes exibem feedbacks claros (carregando, erro, sucesso) e os resultados são apresentados de forma comparativa.

Segurança: O corpo HTML retornado é exibido com dangerouslySetInnerHTML apenas para visualização; em produção, recomenda-se sanitizar (ex: DOMPurify) para evitar XSS.
