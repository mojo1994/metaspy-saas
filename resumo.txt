## Schema do Banco
```sql

  id TEXT PRIMARY KEY, nome TEXT, email TEXT UNIQUE, senha TEXT,
  plano TEXT DEFAULT 'nenhum', plano_expira TIMESTAMP,
  email_verified INTEGER DEFAULT 0, created_at TIMESTAMP, updated_at TIMESTAMP

  type TEXT DEFAULT 'page', published INTEGER DEFAULT 1,
  created_at TIMESTAMP, updated_at TIMESTAMP
```
